View on GitHub How it works 
# Clone and start fresh
$ git clone <this-repo> my-project
$ cd my-project && rm -rf .git && git init

# Install the ECC plugin
$ claude
> /plugin marketplace add affaan-m/everything-claude-code
> /plugin install everything-claude-code@everything-claude-code

# Install rules for your language
$ git clone affaan-m/everything-claude-code /tmp/ecc
$ cd /tmp/ecc && ./install.sh <language>
#   options: typescript | python | golang | swift

# Restart and build
$ claude
Any stack

Tell Claude your stack and it reconfigures everything

The template ships with a Next.js + Supabase default, but it adapts to anything. Claude rewrites CLAUDE.md, conventions, commands, and project structure in seconds.

"I'm building a Go API with HTMX and SQLite. Update CLAUDE.md for this stack."
"This is a Python FastAPI project with PostgreSQL. Reconfigure everything."

Just describe your stack. Claude handles the rest.

How it works

Two layers, one setup

This template

Project-specific layer

Your project's memory, conventions, and task tracking. Committed to git, shared with your team.

  • CLAUDE.md — stack, architecture, commands, conventions
  • TASKS.md — sprint tracker with session continuity
  • .gitignore — sensible defaults
ECC Plugin

Agent toolkit layer

Production-grade agents, skills, hooks, and rules. Installed as a plugin, managed separately.

  • 16 agents — planner, architect, TDD, code review, security
  • 65+ skills — /plan, /tdd, /security-scan, /e2e
  • 40+ commands — slash commands for common workflows
  • Per-language rules — TypeScript, Python, Go, Swift
Adapt to any stack

Four examples, infinite possibilities

The default is Next.js + shadcn + Supabase. Here's how the same template adapts to completely different stacks.

Next.js + shadcn + Supabase

App Router, Server Components, Tailwind, RLS, pnpm
"This is the default. Just fill in your project name and start building."
Rules: ./install.sh typescript

Python + FastAPI + PostgreSQL

SQLAlchemy 2.0, Pydantic v2, Alembic, uv, pytest
"Update CLAUDE.md for a FastAPI project with PostgreSQL, SQLAlchemy, and Pydantic. Use uv as package manager."
Rules: ./install.sh python

Go + HTMX + SQLite

stdlib net/http, Go templates, modernc.org/sqlite, goose
"Reconfigure for a Go web app using stdlib, HTMX for the frontend, and SQLite with goose migrations."
Rules: ./install.sh golang

Rails 8 + Hotwire

Turbo, Stimulus, Tailwind, PostgreSQL, Solid Queue
"Change the stack to Rails 8 with Hotwire, PostgreSQL, and Solid Queue for background jobs."
Rules: ./install.sh typescript (for JS/frontend)

What ships in the template

Intentionally minimal. ECC handles the heavy lifting. These files are what ECC doesn't provide.

claude-bootstrap-template/
├── CLAUDE.md                          # project memory — loaded every session
├── TASKS.md                           # sprint tracker — read at session start
├── .env.example                       # required environment variables
├── .gitignore                         # sensible defaults for web projects
├── LICENSE                            # MIT
└── docs/
    └── index.html                    # this page (GitHub Pages)
