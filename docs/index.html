Use in CI

Once your config scores well, lock it in. Add this GitHub Action to your repo and every pull request that touches Claude Code config files will be checked automatically. If someone's change drops the score below your threshold, the PR gets blocked until they fix it. Your config quality never degrades.

# .github/workflows/claude-config.yml
name: Claude Code Config Quality
on:
  pull_request:
    paths: ['CLAUDE.md', '.claude/**', '.claudeignore']
jobs:
  config-check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: '22' }
      - run: npx claude-launchpad@latest doctor --min-score 80 --json

Copy this file to your repo. Score below 80 = PR blocked.
