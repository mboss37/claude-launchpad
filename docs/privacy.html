Privacy Policy

Last updated: March 28, 2026

Claude Launchpad is an open source CLI tool and Claude Code plugin. This policy explains what data the tool accesses and what it does not collect.

What the tool reads

Claude Launchpad reads files in your project directory to diagnose and scaffold Claude Code configurations:

  • CLAUDE.md, .claude/, .claudeignore — analyzed by the doctor command
  • package.json, go.mod, pyproject.toml, and other manifest files — used to detect your project's language and framework

All file reading happens locally on your machine. No file contents are sent to any server.

What the tool does NOT collect

  • No telemetry, analytics, or usage tracking of any kind
  • No personal information
  • No file contents sent to external servers
  • No cookies, local storage, or browser tracking on the landing page
  • No account creation or authentication required

Eval command

The eval command runs Claude Code headless in temporary sandboxed directories to test your configuration. This uses the Anthropic API through your existing Claude Code CLI or the Agent SDK. API usage is subject to Anthropic's privacy policy. Claude Launchpad does not independently send data to Anthropic — it uses your local Claude Code installation.

Enhance command

The enhance command spawns an interactive Claude Code session to analyze your codebase. This runs entirely through your local Claude Code CLI. No data passes through Claude Launchpad — it simply launches Claude with a prompt.

Third-party services

  • npm — the package is distributed via npm. npm's privacy policy applies to package downloads.
  • GitHub Pages — the landing page is hosted on GitHub Pages. GitHub's privacy statement applies.
  • Google Fonts — the landing page loads Inter and JetBrains Mono from Google Fonts. Google's privacy policy applies.

Open source

Claude Launchpad is MIT licensed. The complete source code is available at github.com/mboss37/claude-launchpad. You can audit exactly what the tool does.
