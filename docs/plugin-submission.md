### Plugin details

**Plugin name:**
```
claude-launchpad
```

**Plugin description:**
```
Is your Claude Code setup actually good? This plugin tells you. It scans your CLAUDE.md, hooks, rules, and settings, gives you a score out of 100, and auto-fixes what's broken. One command to set up any project. One command to fix common issues. One command to test if Claude actually follows your rules. Works with any language or framework.
```

**Example use cases:**
```
Example 1: You're starting a new project and want Claude Code configured properly from day one. Run /launchpad:init to detect your stack and generate everything.

Example 2: You just joined a team and want to know if their Claude Code setup is any good. Run /launchpad:doctor to score the config and see what's missing.

Example 3: Your CLAUDE.md has TODO placeholders and generic advice. Run /launchpad:enhance so Claude reads your actual codebase and fills in real architecture, conventions, and guardrails.

Example 4: You changed your CLAUDE.md but don't know if it actually makes Claude behave differently. Run /launchpad:eval to test against real scenarios and see if your rules work.
```
